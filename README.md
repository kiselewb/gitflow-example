1. Создать репозиторий на GitHub и клонировать его.
2. Создать ветку разработки (develop) от главной ветки.
3. Создать от ветки "develop" ветку "feature/main-page" и проивести слияние "feature/main-page" в "develop", когда feature будет выполнен.
4. Создать ветки "release/version" от "develop".
5. Когда ветка "release" закончена, нужно произвести слияние в "develop" и "main", а затем удалить ее.
6. Если в ветке "main" обнаруживается ошибка - создается ветка "hotfix".
7. Когда работа над "hotfix" завершается - нужно произвести ее слияние в "develop" и "main".
